## Key Takeaways

- Search skills are essential in cybersecurity.
- Shodan helps find exposed internet-facing services.
- VirusTotal helps analyze suspicious files, URLs, domains, and IPs.
- CVE databases help research known vulnerabilities.
- MAN pages help understand Linux commands.
- GitHub is useful for finding tools, scripts, and references.
- Documentation is very important in cybersecurity.

## Defensive Learning

- Publicly exposed services should be monitored.
- Outdated software should be patched.
- Suspicious files and URLs should be checked before opening.
- GitHub repositories should not expose secrets.
- System permissions should be reviewed regularly.
